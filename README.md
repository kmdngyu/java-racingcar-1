# java-racingcar
자동차 경주 게임 미션 저장소

목표
1. getter, setter를 사용하지 않음
2. 유효한 테스트 코드 작성
3. 뎁스 2 준수 ( 가능한 1 유지 )

문제점
1. 입력검증 테스트 코드는 private라 작성이 불가능함
2. 복수의 우승자 판별 코드 뎁스 2
